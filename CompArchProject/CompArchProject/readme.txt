Write your binary code in the "unFormatted.txt" file in the following format:
xxxx xxxx xxxx xxxx
xxxx xxxx xxxx xxxx
xxxx xxxx xxxx xxxx
xxxx xxxx xxxx xxxx

Run the "FileFormatter.py" script and it will change the file to this format (it is less read-able)
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx...

You can use "TranslateBinary.py" to convert the "unFormatted.txt" into readable assembly code

LOAD WORD:
lw $rt, immediate($rs) --> $rt = programMemory[$rs+immediate]

STORE WORD:
sw immediate($rs), $rt --> programMemory[$rs+immediate] = $rt
